# Projeto GitOps

## Objetivo

Este projeto tem como objetivo simular um fluxo GitOps para atualização automática de aplicações através de um repositório Git.

## Tecnologias utilizadas

- GitHub
- YAML
- Kubernetes
- ArgoCD
